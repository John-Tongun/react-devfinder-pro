# DevFinder Pro

## Project Overview

DevFinder Pro is a modern Single Page Application (SPA) built with
React.js, React Router, Bootstrap 5, and the GitHub REST API.

The application allows users to search for GitHub developers, view
detailed profile information, explore public repositories, manage a list
of favorite developers, and customize application settings.

This project was designed as an educational exercise to help students
learn the core concepts of modern React development while building a
real-world application that consumes data from an external API.

---

## Learning Objectives

By completing this project, students will learn how to:

* Build modern Single Page Applications (SPAs)
* Create and organize reusable React components
* Manage local component state using the `useState` hook
* Handle side effects using the `useEffect` hook
* Consume data from RESTful APIs
* Implement client-side routing with React Router
* Create dynamic routes using URL parameters
* Extract route parameters using the `useParams` hook
* Build and manage forms using controlled components
* Share state across components using the Context API
* Implement reusable logic using custom hooks
* Apply component composition and separation of concerns
* Handle loading, success, and error states
* Create responsive user interfaces using Bootstrap 5
* Structure React projects using industry-standard practices

---

## Features

### Developer Search

Users can search for GitHub developers by username using the GitHub
Search API.

### Developer Profile

Users can view detailed information about a selected developer,
including:

* Username
* Profile picture
* Biography
* Public repositories

### Repository Explorer

Users can browse repositories associated with a selected GitHub
developer.

### Favorites Management

Users can save developers to a favorites list using the Context API,
demonstrating global state management in React applications.

### Application Settings

Users can modify application preferences through a settings page,
demonstrating form handling and controlled components.

### Responsive Design

The application uses Bootstrap 5 to provide a responsive and
mobile-friendly user interface.

## Educational Purpose

This project serves as a bridge between JavaScript fundamentals and
professional React development. Students will gain hands-on experience
building a real application while learning the tools, patterns, and
practices commonly used in modern frontend development.

The project emphasizes clean code, reusable components, separation of
concerns, and practical problem-solving skills that are directly
applicable to real-world software engineering projects.
